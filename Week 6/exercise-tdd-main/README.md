# Exercises Test-Driven development

## Exercise 1
* With this exercise, you practice the implementation and refactoring of code when tests are given. 
* In the given folder, you find an existing project with a test class for User. 
* Uncomment each test one by one and refactor your code until the test runs green.

## Exercise 2: StudentTest
* Uncomment the code in Student.java. 
* Implement tests for the constructor (one happy case and unhappy case for the studies field: null and empty string). 
* Implement tests for all methods overridden from the User superclass.

## Exercise 3: ParentTest
* Uncomment the code in Parent.java.
* Add an extra field maxNumberOfChildren to the class Parent.
* This field is set when a new Parent is created. The parent cannot have more children than this parameter.
* Implement this functionality according to the principles of TDD: first write tests (for the constructor with the extra field and addChild() ), implement code afterwards.
